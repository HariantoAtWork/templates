## Better-Auth Nuxt 4 Module

> nuxt-modules-auth

A Better-Auth module for Nuxt 4 that can be installed in your `modules` folder. Features proxy support for connecting to a central Better-Auth service, enabling distributed authentication across multiple applications. 

```bash
npx giget@latest --install gh:hariantoatwork/templates/nuxt-modules-auth modules/0000-auth
```

Setting up `.env` or Environmetal Variables are required
