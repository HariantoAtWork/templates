## Nuxt 3

> nuxt-3

A minimal Nuxt 3 starter template for building server-rendered Vue.js applications. Includes Sass support and Docker deployment configuration with build, run, and push scripts ready for containerised deployments.

```bash
npx giget@latest --install gh:hariantoatwork/templates/nuxt-3 myProject
```

## Nuxt 4

> nuxt-4

A feature-rich Nuxt 4 starter template with TypeScript, Nuxt UI components, Content module, Image optimisation, ESLint, and Scripts support. Includes SQLite database integration and uses pnpm as the package manager. Perfect for building modern, content-driven applications.

```bash
npx giget@latest --install gh:hariantoatwork/templates/nuxt-4 myProject
```

## Better-Auth Nuxt 4 Module

> nuxt-modules-auth

A Better-Auth module for Nuxt 4 that can be installed in your `modules` folder. Features proxy support for connecting to a central Better-Auth service, enabling distributed authentication across multiple applications. 

```bash
npx giget@latest --install gh:hariantoatwork/templates/nuxt-modules-auth modules/0000-auth
```

Setting up `.env` or environmental variables are required.

## PrimeVue

> primevue

A Nuxt 3 starter template with PrimeVue UI component library, Tailwind CSS integration, and PrimeIcons. Includes Sass support and Axios for HTTP requests. Ideal for building enterprise-grade applications with a rich component ecosystem and modern styling capabilities.

```bash
npx giget@latest --install gh:hariantoatwork/templates/primevue myProject
```
